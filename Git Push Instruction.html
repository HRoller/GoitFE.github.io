Инструкция и ключевые моменты использования консоли:
1) Копируем себе Репозиторий
git clone https://github.com/HRoller/frontend_test.git
2) Заходим в папку на кампе (уже скопированного с репозитория)
cd frontend_test
3)Показать список всех файлов локального Гита на кампе (директории которой находимся)
ls -la
4) Новое состояние (save) Git на кампе! делаем c помощью команды
git add .
git commit -m "Save 0.1 или другой камент о том что делали в этой версии"
5) Проверить изменения
git log
6) Отсылаем с локального на репозиторий в интернете
git push
Удачи!
